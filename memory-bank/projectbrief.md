# Project Brief: Paint (C++ SFML Project)

This project is a C++ application using the SFML (Simple and Fast Multimedia Library) for graphics and input.

## Core Requirements
- Language: C++17
- Compiler: clang++ (Apple)
- Build System: CMake
- Graphics/Input: SFML 2.6+ (SFML 3.0.0 detected)
- Platform: macOS ARM64 (M1)

## Goals
- Setup a robust C++ development environment on macOS.
- Implement a basic SFML window to verify the setup.
- Provide a foundation for a "Paint" style application.
